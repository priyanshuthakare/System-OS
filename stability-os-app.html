Design Principles
No rounded corners. Sharp edges. This is a system, not a social app.
No icons as decoration. Every element earns its place.
No gradients. Flat, dense, honest.
Monospace for data. Numbers and labels always mono.
Condensed for titles. Maximum impact, minimum space.
3-color semantic system. Red/Amber/Green only.
APP ARCHITECTURE — SCREEN MAP
Tab 1: TODAY — Home state display, active block checklist, score strip, urge button
Tab 2: LOG — Full day schedule detail, all 7 blocks, step timers, progress tracking
Tab 3: STATS — Weekly calibration, rule compliance grid, phase progress, metrics
Tab 4: PROFILE — Phase status, identity log, system settings
Overlay: URGE PROTOCOL — Full-screen takeover, 5-step sequence, countdown timer
Overlay: FAILURE CONTAINMENT — Triggered on relapse tap, diagnostic + re-entry
State Bar — Always visible, time-based state detection (5 states auto-switch)
